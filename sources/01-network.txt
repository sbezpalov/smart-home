# 🌐 Сетевая инфраструктура

## 🖥️ Основное оборудование

| Устройство | Модель | IP-адрес | Роль | Порты/Uplink |
|------------|--------|----------|------|--------------|
| Маршрутизатор | MikroTik RB5009 | 192.168.1.1 | Основной роутер, DHCP, VLAN | Uplink: 1Gbps WAN |
| Коммутатор 1 | TP-Link SG2210P | 192.168.1.2 | PoE коммутатор (Камеры/AP) | 8x PoE+ |
| Коммутатор 2 | Cisco SG250 | 192.168.1.3 | Ядро сети | 24x Gigabit |
| Минисервер | Intel NUC i5 | 192.168.1.10 | Home Assistant, Docker | 2x LAN (LAG) |
| NAS | Synology DS220+ | 192.168.1.20 | Хранилище, NVR | 2x LAN |

## 📦 Сетевые сервисы

- **DHCP**: 192.168.1.100-200
- **DNS**: 192.168.1.1 (основной), 8.8.8.8 (резервный)
- **VLAN ID 10**: Управление (Management)
- **VLAN ID 20**: IoT устройства
- **VLAN ID 30**: Гостевая сеть

## 🔌 План портов (коммутатор 1)
- Port 1: Uplink к роутеру
- Port 2: Минисервер
- Port 3: WiFi AP (Гостиная)
- Port 4: WiFi AP (Кабинет)
- Port 5-8: IP камеры
