# Wiren Board — контроллер IoT

Контроллер **Wiren Board** используется для интеграции со сторонними решениями через **MQTT** и связки с **Home Assistant** (основной инстанс HA — отдельный хост **ch-smart**, см. [Network](01-network.md)).

### Принцип vendor-segregated Zigbee

В системе сознательно держится **по одному Zigbee-координатору на вендора** — это снижает риск неоднородной mesh с Tuya-кластерами (у вендора часто нестандартная реализация).

| Координатор | Контур | Где физически | Назначение |
|-------------|--------|---------------|------------|
| Aqara M3 Pri | Aqara | Boiler Room | основные сценарии Aqara |
| Aqara M3 Sec | Aqara | Kitchen | вторая зона + резервирование |
| Aqara M1S | Aqara | Bathhouse | **координатор** всей Aqara-периферии бани (`54EF44322EBA`) |
| Camera G2H | Aqara | Bathhouse | камера; **самостоятельный** hub (`lumi1.54ef44350ea7`), не координатор для остальных устройств |
| Wiren Board `WBE2R-R-ZIGBEE` | Tuya | Boiler Room | пульт Tuya *(котёл Kiturami — Wi‑Fi/Tuya, см. [`09-devices.md`](09-devices.md))* |

Топики и ACL — *(дополнить по фактической конфигурации)*. Совпадение **10.254.10.5** с нижней границей DHCP-пула VLAN 10 на FortiGate — см. [`01-network.md`](01-network.md).
- **Home Assistant** — на **10.254.10.3** (**ch-smart**); схема: MQTT / discovery / нативные или community-интеграции для Wiren Board — *(уточнить установленные add-on / integration)*.

## LoRaWAN шлюз (Lavritech)

Отдельное устройство **L2 Gate D2-Lora-ETH** ([lavritech.com](https://lavritech.com)), в веб-интерфейсе отображается как **CH-L2-Gate**, режим **Pro**.

| Параметр | Значение |
|----------|----------|
| IP (VLAN 10) | **10.254.10.5** |
| Профиль ПО | **202: L2 Gate D2-Lora-ETH** (каталог / профиль **soft.lavritech**, сборка **10.02.26** на скриншоте выбора приложения) |
| Связь с Wiren Board | **RS-485**, разъём в режиме **Modbus slave** (проводной slave к контроллеру; изолированный RS-485 на WB — слот **WBE2-I-RS485-ISO**) |
| MQTT | Включено **Enable send MQTT**; брокер **`mqtt://10.254.10.2`**, порт **1883**, период **600** с; логин/пароль на скриншоте **пустые** — для продакшена задать учётку и/или ACL на брокере |
| Прочее (Hardware) | **WB connector** — *No use*; **DS18B20** — выкл.; GPIO датчиков на главном экране пустые |

*Uptime и объём свободной памяти на скриншоте меняются со временем — не дублировать как «истину».*
