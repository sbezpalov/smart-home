# Wiren Board — контроллер IoT

Контроллер **Wiren Board** используется для интеграции со сторонними решениями через **MQTT** и связки с **Home Assistant** (основной инстанс HA — отдельный хост **ch-smart**, см. [Network](01-network.md)).

## Сеть

| Параметр | Значение |
|----------|----------|
| IP | **10.254.10.2** |
| Hostname | **ch-wirenboard** |
| VLAN | **10 (IoT)** — `10.254.10.0/24`, шлюз **10.254.10.1** |

## Аппаратные модули (`/etc/wb-hardware.conf`)

| Слот / позиция | Модуль |
|----------------|--------|
| Внутренний слот 1 | **WBE2-I-RS485-ISO** — изолированный **RS-485** |
| Внутренний слот 3 | **WBE2R-R-ZIGBEE** — **ZigBee** |
| Внутренний слот 4 | **WBE3-R-GPS** — **GPS/ГЛОНАСС** |
| Слот модема | **WBC-4G** — **4G+3G+2G** |

*Внутренний слот 2, модули WBIO 1–8, интерфейсы RS485-1/2, режимы W1/W2 на предоставленных снимках не зафиксированы — дополнить при необходимости.*

## Интеграции

- **MQTT** — обмен с периферией и внешними сервисами; брокер, топики и ACL — *(зафиксировать по фактической конфигурации)*.
- **Home Assistant** — на **10.254.10.3** (**ch-smart**); схема: MQTT / discovery / нативные или community-интеграции для Wiren Board — *(уточнить установленные add-on / integration)*.
