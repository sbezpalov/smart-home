# 🎥 Видеонаблюдение и безопасность

## Политика

- Камеры **бренда Hikvision**, заведённые на NVR по **HIKVISION** (порт **8000**), не через «голый» ONVIF. Потоки для VMS — **через NVR** `10.254.50.2` (номера каналов — **Digital Channel** в GUI / iVMS).
- Каналы **D6/D7** в NVR сейчас **ONVIF:80** (камеры **10.254.50.10** / **10.254.50.8**) — не режим HIKVISION; см. страницу **Video and Security** в Notion.
